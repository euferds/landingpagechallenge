# Clipboard Landing Page — Responsive Frontend Challenge

## 🇺🇸 EN

Landing page developed as a solution for the "Clipboard Landing Page" challenge from Frontend Mentor. The project was built with semantic HTML5 and CSS3, using Flexbox, CSS Grid, Media Queries, and a Mobile First approach to create a responsive interface. It also includes accessibility practices, global variables with `:root`, pseudo-classes and pseudo-elements, microinteractions, a custom CSS tooltip, and Google Fonts integration.

This repository contains the `AGENTS.md` and `CLAUDE.md` files automatically included by the Frontend Mentor template for AI assistant integration. During development, AI was used only as support for learning and occasional questions.

Landing page desenvolvida como solução do desafio "Clipboard Landing Page" do Frontend Mentor.
